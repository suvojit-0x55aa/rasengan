# NARUTO

Hand-tracked jutsu activation with Naruto/Sasuke effects.

### How to use:
- Open `index.html` in your browser (webcam required).

### Controls:
- **Fist → Open hand (left):** Triggers Naruto effect (plays once, fades in)
- **Fist → Open hand (right):** Triggers Sasuke effect (plays once, fades in)
- **Open hand → Fist:** Effect fades out
- **Blue skeleton:** Hand tracking visualization

### How it works:
- Animations play **once** per fist-to-open gesture — no looping
- Opening your hand starts the effect with a gradual build-up
- Closing back to a fist (or removing your hand) fades the effect out quickly
- Each hand is independent — trigger both simultaneously for dual jutsu

### Note:
- Had to remove the glow to make it compatible for all devices.
